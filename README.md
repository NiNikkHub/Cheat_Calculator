# Калькулятор ТВО / МС

Next.js-сервис с научным калькулятором и вкладкой «Билеты и ответы».

## Что умеет

- вычисления с операциями `+`, `−`, `×`, `÷`, скобками, степенями и процентами;
- `sin`, `cos`, `tan`, `√`, `log`, `ln`, `π`, `e`;
- переключение градусов и радиан;
- память `MC`, `MR`, `M+`, `M−` и история вычислений;
- 30 экзаменационных билетов с двумя теоретическими вопросами и практикой;
- билеты №17–30 помечены как дубликаты билетов №1–14.

## Запуск

```powershell
npm install
npm run dev
```

Откройте адрес, который выведет терминал, обычно `http://localhost:3000`.
